# ChatApp

## Setup
```
yarn dev
```
```
yarn start
```
```
yarn storybook
```
```
yarn cypress
```
```
Environment variables:
- MONGOURL - mongodb url,
- NEXT_PUBLIC_HOST - page address
```
Admin privileges can only be set by editing the user role in the database.
